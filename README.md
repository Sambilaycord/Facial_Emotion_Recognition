# Facial Emotion Recognition

This project focuses on detecting human emotions from facial images using the **FER2013** dataset. It explores various machine learning and deep learning approaches, including Convolutional Neural Networks (CNN) and landmark-based classification.

### Key Features
- **Dataset**: FER2013 (7 emotion classes: angry, disgust, fear, happy, sad, surprise, neutral).
- **Models**: MobileNetV2 (CNN features), Random Forest, and Logistic Regression.
- **Technologies**: Python, Keras/TensorFlow, OpenCV, Scikit-learn, and MediaPipe.

### Directory Structure
- `main.ipynb`: Core Jupyter notebook containing data preprocessing, model training, and evaluation.
- `data_test/`: Directory containing sample images for inference testing.
